How We Can Find Any Device From Ip Address?

An IP address is assigned to each device that is connected to the Internet by the Internet Service Provider. IPv4, whose IP addresses are quite widely used today,has a size of 32 bits. In this regard, it is indicated by 4 8-bit numbers separated by dots.

For example: 192.168.10.5 (the IP address given in this example is defined as a private IP address. At the same time, only local networks can provide communication. In order to communicate with other networks, it is necessary that the device has an IP address for general (public) use. Also from the ip address we can not find the exact location of the device, we get a close location.

So how do we find it?

We will use a ip-tracer tool for this and open the terminal. Then you will need to enter the following commands in order to install the ip-tracer tool.

' apt-get update '
' apt install git curl -y '
' cd desktop/ '
' git clone https://github.com/htr-tech/track-ip.git '

After installing it, we go into the tool, and then enter the following commands;

' cd track-ip '

With the ' bash trackip ' command, we also run the tool.
