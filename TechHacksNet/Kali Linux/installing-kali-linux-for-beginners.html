Installing Kali Linux from Scratch - Downloading Kali Linux (2022)

- Installing Kali Linux from Scratch - Downloading Kali Linux to a Computer
Before downloading Kali Linux, you need to install a virtual machine on the computer. For this purpose, you will need to download VirtualBox. You can also install it directly on USB without installing a virtual machine, but the virtual machine is more preferable because it is easier to access kali linux with a virtual machine.

Kali Linux System Requirements

The installation of Kali Linux depends on the features and privileges that you have set, but the minimum criteria that should be present on your computer system;

  • RAM: 2048 MB RAM
  • Required Storage Space: At least 16 GB

Requirements For Installation

  1. An ISO file. for installing Kali Linux.
    • On the site that opens, select the one for VirtualBox.
    • If your computer supports it, select the 64-Bit version.
  2. Then you will need to do the installation of the virtual machine
    • You can download VirtualBox here
  3. After entering the site, you can click on the Download icon and install it on your computer.
  4. After installing the virtual machine on your computer, follow the steps to install kali linux.
    • You are opening the first VirtualBox.
    • Then we click on the "File" located in the upper right part.
    • One of the options that comes up after clicking on "Import Appliance.." we say.
    • Then the "Source" on the screen remains the Local File System, and you select the Kali Linux file that you downloaded earlier in the file section. After selecting it, we click on the "Next" button.
    • Then, without changing anything, we click on "Import" and click on the "Agree" button.

      • After doing all this, you will have installed Kali Linux on your computer. After that, you can make the settings according to your own computer.
