Kali Linux Main Commands

We have compiled the main commands of the Kali Linux system and what they do for you;

The apt-get Command

The apt-get command is a command used to perform operations such as downloading, installing dependencies, upgrading, and updating new packages from defined package sources used in Linux distributions.

For example;

' apt-get update && upgrade '

By saying so, you will be updating and upgrading your system at the same time.

The cat Command

The Cat command is most often used to dump the contents of a text file onto the screen. Of course, it is possible to perform this operation directly in such a way that the text itself or part of it is selected. In addition to this process, it also performs the of saving the contents of different text files to another file. If we combine it with the previous sentence, we can say that it is possible to create a new text file content from texts to be selected from different text files.

For example;

' cat file.txt '

The pwd Command

It is used to find out in which directory it is located at the time of the transaction.

The ls Command

The ls command is used to show the files that exist in the directory in which we are located.

The cd Command

Linux also allows us to navigate through its files. It requires either the exact location or the name of the directory, depending on the current directory you are in.For example, we need to specify which path we need to go to between these icons (/), respectively. And if so, how many levels must be returned from this directory cd .. by doing this, we can go from that directory to the previous directory. If we need to go to the root directory, we need to do cd ~.

For example;

' cd Desktop/ '

The mv Command

The mv command is used to move the file. To explain it more clearly, if we transfer the file in the directory located in Linux systems to the directory we want to move it to.

For example;

' mv file.txt /root/Desktop '

The mkdir Command

The mkdir command is used to create a new directory.

For example;

' mkdir your_file '

The rmdir Command

We use the rmdir command to delete the previously created directory. to use the rmdir command, the inside of this directory must be empty.

The rm Command

The rm command does not need to be empty when we delete a file or to delete a directory such as the rmdir command.

For example'

' rm -rf Videos '

The sudo Command

The purpose of using the sudo command is to exceed the user's authority when performing some operations, but with the help of the sudo command, if this user is attached to the sudoers file, this user can perform operations at the root level.

The locate Command

The locate command is the Decode command. If we need to explain more clearly, how we perform a search operation to find the command we want to search for on other operating systems, Linux also has more than one form of search, the locate command is one of them. when using the locate command, there is a case sensitivity to avoid this, the locate -i command is used.

For example;

' locate -i which_you_wan't_to_find_file '

The grep Command

We often use the grep command in the Linux operating system.The grep command is used to search for a specific part in the text or in another file.
